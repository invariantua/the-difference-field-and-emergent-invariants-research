# Difference Fields

Research workspace for texts, notes, figures, references, and toy models related to 
difference fields, compression, emergent invariants, space, time, and self.

## Structure

- `docs/` — roadmap, glossary, assumptions, referee notes
- `papers/` — LaTeX manuscripts
- `notes/` — daily notes, ideas, literature notes
- `models/` — toy agents, simulations, notebooks
- `figures/` — source and exported figures
- `references/` — bibliography and stored papers
- `scripts/` — helper scripts

## Quick start

Build the position paper PDF:

```bash
make paper
```
