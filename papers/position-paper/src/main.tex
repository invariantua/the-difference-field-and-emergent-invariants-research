\begin{abstract}
This paper proposes a conceptual and formal framework in which the primitive level of
description is not a set of objects in space and time, but a domain of distinctions.
Objects are treated as compression-relative invariants, while temporal, spatial, and
self-related structures are discussed as emergent organizations of constrained representation.
\end{abstract}
